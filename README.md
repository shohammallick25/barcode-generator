# Barcode Generator
<img width="1877" height="877" alt="image" src="https://github.com/user-attachments/assets/de147297-6255-4888-843c-65283fea15c1" />

A simple web app to generate barcodes from input text using JsBarcode library.

## Features

- Generates barcode dynamically from user input
- Download barcode as PNG image
- Share barcode image via supported devices (using Web Share API)
- Futuristic neon style UI with background video

## How to Use

1. Enter any text in the input field.
2. Click the **Barcode Generator** button.
3. Your barcode will appear below.
4. Use **Download** to save as PNG or **Share** to share the image.

## Technologies Used

- HTML5, CSS3, JavaScript
- [JsBarcode](https://github.com/lindell/JsBarcode) for barcode generation
